# Customer Churn Prediction

## Project Overview
This project predicts whether a customer will churn or not using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Models Used
- Logistic Regression
- Random Forest

## Results
- Logistic Regression Accuracy: 78.54%
